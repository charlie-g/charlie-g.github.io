PROJECTS

. General Assembly .

READR - Newsreader

Concept: Created as a first project for the WDI course at General Assembly, readR is a newsreader allowing users to add their own rss feed and make them directly available to other users to make it easier to discover new websites.

Toolkit: This project is a Rails app and uses Bootstrap in the front-end.

LOOPLE - An Interactive Music Composer

Concept: As a second project, two other students and I created Loople, an interactive music composer in Javascript. This website uses the keyboard to generate sounds and animations on the screen.

Toolkit: This project is a Rails app and uses Javascript and the Web Audio API.

SPECTRM - A Music Visualizer

Concept: My third and final project is a music visualization app. The 3D shapes move according to the sound playing and you can either use the sound pre-loaded, the microphone or any song from Soundcloud.

Toolkit: This project is a Rails app and uses Three.js, the Web Audio API, the Soundcloud API and the Leap Motion API.
