Loople

A music composer in your browser.

Loople is a website that allows you to play music using your computer keyboard. Each key plays a sound and an animation and you can save your loops by signing up.

  • Front-End: HTML, CSS, jQuery
  • Back-End: Ruby, Rails, Web Audio API

Check out Loople here.

Spectrm

A music visualizer.

Spectrm is a wesbite that allows you to visualize your favorite tracks. Either using the pre-selected song, your computer's microphone or any link from Soundcloud, choose any of the 3 visualizations proposed to see the graphics interact with the sound.

  • Front-End: HTML, CSS, jQuery, Three.js
  • Back-End: Ruby, Rails, Web Audio API, Soundcloud API

Check out Spectrm here.

Brainwaves Visualiser

A visualizer for the neurosky headset.

The Neurosky visualizer is an experiment to try and identify patterns in brainwaves when executing eye movements with the Neursosky headset. If this is successful, the next step would be to use these patterns to control device with your mind.

  • Front-End: HTML, CSS, jQuery, d3.js
  • Back-End: Node.js

Work still in progress, this visualizer should be live soon!

Sphero + Leap Motion

Control the Sphero using the Leap Motion

Using Node.js, this project aims at using your hand movements over the Leap Motion to apply directions to the Sphero robotic ball.

At the moment, Up & Down will stop the ball while forward, backward, left and right apply these directions to the Sphero.

You can view the source code on my Github.

Sphero + Myo Armband

Control the Sphero using the Myo armband

The Myo armband is an armband that analyses the activity of the muscles in your arm and can detect certain gestures or orientations. Paired with the Sphero using Node.js & Cylon.js, I managed to make the Sphero roll depending on which gestures I was executing wearing the armband.

You can view the source code on my Github.

Drone + Leap Motion

Control the Parrot AR drone with the Leap Motion

Using the Leap Motion controller and Node.js, I managed to make the drone fly depending on which movement I execute over the Leap Motion.

You can view the source code on my Github.

Drone + Neurosky

Control the Parrot AR Drone with the Neurosky headset

The Neurosky brainsensor is a headset that collects in real time the data comming back from your brain activity. Using this data, I managed to make my drone take off by focusing enough. This project is still in progress as I would like to use my Neurosky visualizer to identify patterns in my brain waves when I move my eyes. These patterns could then be used to control the directions of the Drone using my eyes.
