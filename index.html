PROJECTS

GA Projects

READR - Newsreader

As a first project for the WDI course at General Assembly, I created a newsreader that would allow users to add their own rss feed and make them directly available to other users so it would be easier to discover new websites.
This project needs to be fixed a little bit.

LOOPLE - An Interactive Music Composer

As a second project, two other students and I created Loople, an interactive music composer in Javascript. This website uses the keyboard to generate sounds and animations on the screen.
V.2 should allow to save all the loops into a full track and let users upload their own sounds.

SPECTRM - A Music Visualizer

My third and final project is a music visualization app mixing the Web Audio API and Three.js. The 3D shapes move according to the sound playing and you can either use the sound pre-loaded, the microphone or any song from Soundcloud.
Some features and testing need to be added.
